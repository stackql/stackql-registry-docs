## `SELECT` examples

Lists all of the available peering locations for the specified kind of peering.


```sql
SELECT
id,
name,
kind,
properties,
type
FROM azure.peering.locations
WHERE kind = '{{ kind }}'
AND subscriptionId = '{{ subscriptionId }}';
```
