## `SELECT` examples

List all grants on the role


```sql
SELECT
name,
created_on,
grant_option,
granted_by,
granted_by_role_type,
granted_on,
granted_to,
grantee_name,
privilege
FROM snowflake.role.grants_on
WHERE name = '{{ name }}'
AND endpoint = '{{ endpoint }}';
```
