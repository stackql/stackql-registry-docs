## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Permissions

To operate on the <code>location_s3</code> resource, the following permissions are required:

### Read
<pre>
datasync:DescribeLocationS3,
datasync:ListTagsForResource</pre>

### Update
<pre>
datasync:DescribeLocationS3,
datasync:ListTagsForResource,
datasync:TagResource,
datasync:UntagResource</pre>

### Delete
<pre>
datasync:DeleteLocation</pre>


## Example
```sql
SELECT
region,
s3_config,
s3_bucket_arn,
subdirectory,
s3_storage_class,
tags,
location_arn,
location_uri
FROM awscc.datasync.location_s3
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;LocationArn&gt;'
```
