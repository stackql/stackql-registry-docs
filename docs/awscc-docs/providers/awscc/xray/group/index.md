## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
filter_expression,
group_name,
group_ar_n,
insights_configuration,
tags
FROM awscc.xray.group
WHERE region = 'us-east-1'
AND data__Identifier = '{GroupARN}';
```
