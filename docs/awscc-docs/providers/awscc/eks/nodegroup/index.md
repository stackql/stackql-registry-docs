## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Permissions

To operate on the <code>nodegroup</code> resource, the following permissions are required:

### Read
<pre>
eks:DescribeNodegroup</pre>

### Delete
<pre>
eks:DeleteNodegroup,
eks:DescribeNodegroup</pre>

### Update
<pre>
iam:GetRole,
iam:PassRole,
eks:DescribeNodegroup,
eks:DescribeUpdate,
eks:ListUpdates,
eks:TagResource,
eks:UntagResource,
eks:UpdateNodegroupConfig,
eks:UpdateNodegroupVersion</pre>


## Example
```sql
SELECT
region,
ami_type,
capacity_type,
cluster_name,
disk_size,
force_update_enabled,
instance_types,
labels,
launch_template,
nodegroup_name,
node_role,
release_version,
remote_access,
scaling_config,
subnets,
tags,
taints,
update_config,
version,
id,
arn
FROM awscc.eks.nodegroup
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;Id&gt;'
```
