## Example
```sql
SELECT
region,
arn,
name,
state,
recording_reconnect_window_seconds,
destination_configuration,
tags,
thumbnail_configuration,
rendition_configuration
FROM awscc.ivs.recording_configuration
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;Arn&gt;'
```
