## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
arn,
creation_time,
id,
max_cpus,
max_gpus,
max_duration,
max_runs,
name,
tags
FROM awscc.omics.run_group
WHERE region = 'us-east-1'
AND data__Identifier = '{Id}';
```
