## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
channel_name,
policy
FROM awscc.mediatailor.channel_policy
WHERE region = 'us-east-1'
AND data__Identifier = '{ChannelName}';
```
