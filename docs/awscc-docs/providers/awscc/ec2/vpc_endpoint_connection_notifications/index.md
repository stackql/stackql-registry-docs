## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
vpc_endpoint_connection_notification_id
FROM awscc.ec2.vpc_endpoint_connection_notifications
WHERE region = 'us-east-1'
```
