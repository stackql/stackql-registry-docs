## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
v_pn_gateway_id,
amazon_side_asn,
tags,
type
FROM awscc.ec2.vpn_gateway
WHERE data__Identifier = '<VPNGatewayId>';
```

## Permissions

To operate on the <code>vpn_gateway</code> resource, the following permissions are required:

### Read
```json
ec2:DescribeVpnGateways
```

### Update
```json
ec2:CreateTags,
ec2:DeleteTags
```

### Delete
```json
ec2:DeleteVpnGateway
```
