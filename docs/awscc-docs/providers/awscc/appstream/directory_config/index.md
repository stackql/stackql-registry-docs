## Example
```sql
SELECT
region,
organizational_unit_distinguished_names,
service_account_credentials,
directory_name,
certificate_based_auth_properties
FROM awscc.appstream.directory_config
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;DirectoryName&gt;'
```
