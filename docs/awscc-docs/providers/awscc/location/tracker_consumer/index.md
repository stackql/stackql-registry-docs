## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
consumer_arn,
tracker_name
FROM awscc.location.tracker_consumer
WHERE region = 'us-east-1'
AND data__Identifier = '{TrackerName}';
AND data__Identifier = '{ConsumerArn}';
```
