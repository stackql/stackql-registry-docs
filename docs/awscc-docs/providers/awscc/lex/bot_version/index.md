## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
bot_id,
bot_version,
description,
bot_version_locale_specification
FROM awscc.lex.bot_version
WHERE data__Identifier = '<BotId>|<BotVersion>';
```
