## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
arn,
name,
display_name,
chat_channel,
engagements,
actions,
integrations,
tags,
incident_template
FROM awscc.ssmincidents.response_plan
WHERE region = 'us-east-1'
AND data__Identifier = '{Arn}';
```
