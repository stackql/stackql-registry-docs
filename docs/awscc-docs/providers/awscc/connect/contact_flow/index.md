## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
instance_arn,
contact_flow_arn,
name,
content,
description,
state,
type,
tags
FROM awscc.connect.contact_flow
WHERE region = 'us-east-1'
AND data__Identifier = '{ContactFlowArn}';
```
