## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Permissions

To operate on the <code>groups</code> resource, the following permissions are required:

### Create
<pre>
iam:CreateGroup,
iam:PutGroupPolicy,
iam:AttachGroupPolicy,
iam:GetGroupPolicy,
iam:GetGroup</pre>

### List
<pre>
iam:ListGroups</pre>


## Example
```sql
SELECT
region,
group_name
FROM awscc.iam.groups

```
