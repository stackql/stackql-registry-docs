## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
resource_set_name,
resources,
resource_set_arn,
resource_set_type,
tags
FROM awscc.route53recoveryreadiness.resource_set
WHERE data__Identifier = '<ResourceSetName>';
```
