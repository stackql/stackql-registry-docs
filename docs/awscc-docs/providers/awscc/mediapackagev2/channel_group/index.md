## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
arn,
channel_group_name,
created_at,
description,
egress_domain,
modified_at,
tags
FROM awscc.mediapackagev2.channel_group
WHERE data__Identifier = '<Arn>';
```
