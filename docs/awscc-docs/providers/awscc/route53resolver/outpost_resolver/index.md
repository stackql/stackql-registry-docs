## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
id,
creator_request_id,
name,
arn,
outpost_arn,
preferred_instance_type,
status,
status_message,
instance_count,
creation_time,
modification_time,
tags
FROM awscc.route53resolver.outpost_resolver
WHERE region = 'us-east-1'
AND data__Identifier = '{Id}';
```
