## Example
```sql
SELECT
region,
arn,
collaboration_arn,
collaboration_identifier,
tags,
analysis_parameters,
analysis_template_identifier,
description,
membership_arn,
membership_identifier,
name,
schema,
source,
format
FROM awscc.cleanrooms.analysis_template
WHERE data__Identifier = '<AnalysisTemplateIdentifier>|<MembershipIdentifier>';
```
