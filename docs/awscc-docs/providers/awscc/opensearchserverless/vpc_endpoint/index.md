## Example
```sql
SELECT
region,
id,
name,
security_group_ids,
subnet_ids,
vpc_id
FROM awscc.opensearchserverless.vpc_endpoint
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;Id&gt;'
```
