## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
subnet_group_name,
description,
subnet_ids,
tags,
a_rn
FROM awscc.memorydb.subnet_group
WHERE region = 'us-east-1'
AND data__Identifier = '{SubnetGroupName}';
```
