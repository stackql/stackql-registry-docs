## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
parameters,
timeout,
job_definition_name,
propagate_tags,
platform_capabilities,
eks_properties,
type,
node_properties,
scheduling_priority,
container_properties,
id,
retry_strategy,
tags
FROM aws.batch.job_definition
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;Id&gt;'
```
