## `SELECT` Example
```sql
SELECT
region,
arn,
description,
documentation_url,
module_name,
module_package,
is_default_version,
schema,
time_created,
version_id,
visibility
FROM aws.cloudformation.module_version
WHERE data__Identifier = '<Arn>';
```

To operate on the <code>module_version</code> resource, the following permissions are required:
