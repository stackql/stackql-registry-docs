## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
<pre>
SELECT *<br/>FROM aws.cloudformation.stack_set<br/>WHERE region = 'us-east-1'<br/>AND data__Identifier = '&lt;StackSetId&gt;'
