## `SELECT` examples
Gets all <code>rules</code> in a region.
```sql
SELECT
region,
event_bus_name,
event_pattern,
schedule_expression,
description,
state,
targets,
arn,
role_arn,
name
FROM aws.events.rules
WHERE region = 'us-east-1';
```
Gets all properties from an individual <code>rule</code>.
```sql
SELECT
region,
event_bus_name,
event_pattern,
schedule_expression,
description,
state,
targets,
arn,
role_arn,
name
FROM aws.events.rules
WHERE region = 'us-east-1' AND data__Identifier = '<Arn>';
```

## `DELETE` example

```sql
/*+ delete */
DELETE FROM aws.events.rules
WHERE data__Identifier = '<Arn>'
AND region = 'us-east-1';
```

## Permissions

To operate on the <code>rules</code> resource, the following permissions are required:

### Read
```json
iam:PassRole,
events:DescribeRule,
events:ListTargetsByRule
```

### Create
```json
iam:PassRole,
events:DescribeRule,
events:PutRule,
events:PutTargets
```

### Update
```json
iam:PassRole,
events:DescribeRule,
events:PutRule,
events:RemoveTargets,
events:PutTargets
```

### List
```json
events:ListRules
```

### Delete
```json
iam:PassRole,
events:DescribeRule,
events:DeleteRule,
events:RemoveTargets,
events:ListTargetsByRule
```
