## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
name,
arn,
secret_arn,
description,
authorization_type,
auth_parameters
FROM aws.events.connection
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;Name&gt;'
```
