## `SELECT` examples
Expands tags for all <code>apis</code> in a region.
```sql
SELECT
region,
api_id,
api_arn,
name,
owner_contact,
dns,
event_config,
tag_key,
tag_value
FROM aws.appsync.api_tags
WHERE region = 'us-east-1';
```
