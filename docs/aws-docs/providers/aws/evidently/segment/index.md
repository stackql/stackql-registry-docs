## `SELECT` Example
```sql

region,
arn,
name,
description,
pattern,
tags
FROM aws.evidently.segment
WHERE data__Identifier = '<Arn>';
```

```json
evidently:GetSegment,
evidently:ListTagsForResource
```

### Delete
```json

evidently:GetSegment,
evidently:UntagResource
```
