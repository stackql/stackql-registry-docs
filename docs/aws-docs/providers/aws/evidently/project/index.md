## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
arn,
name,
description,
data_delivery,
app_config_resource,
tags
FROM aws.evidently.project
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;Arn&gt;'
```
