## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
name,
description,
app_arn,
resiliency_policy_arn,
tags,
app_template_body,
resource_mappings,
app_assessment_schedule
FROM aws.resiliencehub.app
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;AppArn&gt;'
```
