## `SELECT` examples
Gets all <code>location_hdfs</code> in a region.
```sql
SELECT
region,
name_nodes,
block_size,
replication_factor,
kms_key_provider_uri,
qop_configuration,
authentication_type,
simple_user,
kerberos_principal,
kerberos_keytab,
kerberos_krb5_conf,
tags,
agent_arns,
subdirectory,
location_arn,
location_uri
FROM aws.datasync.location_hdfs
WHERE region = 'us-east-1';
```
Gets all properties from an individual <code>location_hdf</code>.
```sql
SELECT
region,
name_nodes,
block_size,
replication_factor,
kms_key_provider_uri,
qop_configuration,
authentication_type,
simple_user,
kerberos_principal,
kerberos_keytab,
kerberos_krb5_conf,
tags,
agent_arns,
subdirectory,
location_arn,
location_uri
FROM aws.datasync.location_hdfs
WHERE region = 'us-east-1' AND data__Identifier = '<LocationArn>';
```

## `DELETE` example

```sql
/*+ delete */
DELETE FROM aws.datasync.location_hdfs
WHERE data__Identifier = '<LocationArn>'
AND region = 'us-east-1';
```
