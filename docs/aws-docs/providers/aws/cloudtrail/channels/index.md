## `SELECT` Example
```sql
SELECT
region,
channel_arn
FROM aws.cloudtrail.channels
WHERE region = 'us-east-1'
```

## Permissions

To operate on the <code>channels</code> resource, the following permissions are required:

### Create
```json
CloudTrail:CreateChannel,
CloudTrail:AddTags
```

### List
```json
CloudTrail:ListChannels
```
