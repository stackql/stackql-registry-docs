## `SELECT` Example
```sql
SELECT
region,
tag_key
FROM aws.lakeformation.tags
WHERE region = 'us-east-1'
```

## Permissions

To operate on the <code>tags</code> resource, the following permissions are required:

### Create
```json
lakeformation:CreateLFTag
```

### List
```json
lakeformation:ListLFTags
```
