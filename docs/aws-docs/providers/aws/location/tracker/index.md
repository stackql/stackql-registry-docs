## `SELECT` Example
```sql
SELECT
region,
create_time,
description,
event_bridge_enabled,
kms_key_enable_geospatial_queries,
kms_key_id,
position_filtering,
pricing_plan,
pricing_plan_data_source,
tags,
tracker_arn,
tracker_name,
update_time,
arn
FROM aws.location.tracker
WHERE region = 'us-east-1' AND data__Identifier = '<TrackerName>';
```


## Permissions

To operate on the <code>tracker</code> resource, the following permissions are required:

### Read
```json
geo:DescribeTracker,
kms:DescribeKey
```

### Update
```json
geo:CreateTracker,
geo:DescribeTracker,
geo:TagResource,
geo:UntagResource,
kms:DescribeKey,
kms:CreateGrant,
geo:UpdateTracker
```
