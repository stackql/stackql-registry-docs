## Methods

## Example
```sql
SELECT
region,
create_time,
description,
kms_key_id,
pricing_plan,
pricing_plan_data_source,
position_filtering,
tracker_arn,
arn,
tracker_name,
update_time
FROM aws.location.tracker
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;TrackerName&gt;'
```
