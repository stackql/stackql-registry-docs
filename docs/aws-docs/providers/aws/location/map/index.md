## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
configuration,
create_time,
data_source,
description,
map_arn,
arn,
map_name,
pricing_plan,
update_time
FROM aws.location.map
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;MapName&gt;'
```
