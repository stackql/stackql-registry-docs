## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
extra_jars_s3_path,
public_key,
number_of_nodes,
arguments,
subnet_id,
public_keys,
security_group_ids,
role_arn,
worker_type,
endpoint_name,
glue_version,
extra_python_libs_s3_path,
security_configuration,
id,
number_of_workers,
tags
FROM aws.glue.dev_endpoint
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;Id&gt;'
```
