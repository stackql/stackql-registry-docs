## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
type,
start_on_creation,
description,
actions,
event_batching_condition,
workflow_name,
schedule,
id,
tags,
name,
predicate
FROM aws.glue.trigger
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;Id&gt;'
```
