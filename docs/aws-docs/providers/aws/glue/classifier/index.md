## Example
```sql
SELECT
region,
id,
x_ml_classifier,
json_classifier,
csv_classifier,
grok_classifier
FROM aws.glue.classifier
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;Id&gt;'
```
