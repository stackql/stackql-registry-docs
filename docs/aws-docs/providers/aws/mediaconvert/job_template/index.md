## Example
```sql
SELECT
region,
category,
description,
acceleration_settings,
priority,
status_update_interval,
settings_json,
id,
arn,
queue,
hop_destinations,
tags,
name
FROM aws.mediaconvert.job_template
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;Id&gt;'
```
