## `SELECT` Example
```sql
SELECT
region,
rest_api_id,
stage_name
FROM aws.apigateway.stages
WHERE region = 'us-east-1'
```

## Permissions

To operate on the <code>stages</code> resource, the following permissions are required:

### Create
```json
apigateway:POST,
apigateway:PATCH,
apigateway:GET,
apigateway:PUT
```

### List
```json

```
