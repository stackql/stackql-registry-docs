## `SELECT` Example
```sql
SELECT
region,
rest_api_id,
resource_id
FROM aws.apigateway.resources
WHERE region = 'us-east-1'
```

## Permissions

To operate on the <code>resources</code> resource, the following permissions are required:

### Create
```json
apigateway:POST
```

### List
```json
apigateway:GET
```
