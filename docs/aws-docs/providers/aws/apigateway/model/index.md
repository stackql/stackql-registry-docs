## `SELECT` Example
```sql
SELECT
region,
content_type,
description,
name,
rest_api_id,
schema
FROM aws.apigateway.model
WHERE data__Identifier = '<RestApiId>|<Name>';
```

## Permissions

To operate on the <code>model</code> resource, the following permissions are required:

### Read
```json
apigateway:GET
```

### Update
```json
apigateway:PATCH,
apigateway:GET
```

### Delete
```json
apigateway:GET,
apigateway:DELETE
```
