## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
content_type,
description,
name,
rest_api_id,
schema
FROM aws.apigateway.model
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;RestApiId&gt;'
AND data__Identifier = '&lt;Name&gt;'
```
