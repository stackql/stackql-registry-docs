## `SELECT` Example
```sql
SELECT
region,
rest_api_id,
name
FROM aws.apigateway.models
WHERE region = 'us-east-1'
```

## Permissions

To operate on the <code>models</code> resource, the following permissions are required:

```json
apigateway:POST,
apigateway:GET
```
