## `SELECT` Example
```sql
SELECT
region,
id,
cloud_watch_role_arn
FROM aws.apigateway.account
WHERE region = 'us-east-1' AND data__Identifier = '<Id>';
```


## Permissions

To operate on the <code>account</code> resource, the following permissions are required:

### Read
```json
apigateway:GET
```

### Update
```json
apigateway:PATCH,
iam:GetRole,
iam:PassRole
```
