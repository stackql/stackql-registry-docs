## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
engine_version,
description,
revision,
authentication_strategy,
engine_type,
data,
id,
arn,
tags,
name
FROM aws.amazonmq.configuration
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;Id&gt;'
```
