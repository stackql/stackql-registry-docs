## `SELECT` Example
```sql
SELECT
region,
arn
FROM aws.inspectorv2.filters
WHERE region = 'us-east-1'
```

## Permissions

To operate on the <code>filters</code> resource, the following permissions are required:

### Create
```json
inspector2:CreateFilter,
inspector2:ListFilters
```

### List
```json
inspector2:ListFilters
```
