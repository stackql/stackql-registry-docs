## Example
```sql
SELECT
region,
description,
resource_access_role,
vpc_config,
secondary_sources,
encryption_key,
secondary_artifacts,
source,
name,
logs_config,
service_role,
queued_timeout_in_minutes,
secondary_source_versions,
tags,
source_version,
triggers,
artifacts,
badge_enabled,
file_system_locations,
environment,
concurrent_build_limit,
visibility,
id,
arn,
build_batch_config,
timeout_in_minutes,
cache
FROM aws.codebuild.project
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;Id&gt;'
```
