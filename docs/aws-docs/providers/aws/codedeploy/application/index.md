## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
application_name,
compute_platform,
tags
FROM aws.codedeploy.application
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;ApplicationName&gt;'
```
