## Example
```sql
SELECT
region,
role_attached_at,
latest_version_arn,
id,
arn,
role_arn,
name,
initial_version,
tags
FROM aws.greengrass.group
WHERE region = 'us-east-1'
AND data__Identifier = '<Id>'
```
