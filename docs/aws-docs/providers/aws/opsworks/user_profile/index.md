## Example
```sql
SELECT
region,
id,
ssh_username,
allow_self_management,
iam_user_arn,
ssh_public_key
FROM aws.opsworks.user_profile
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;Id&gt;'
```
