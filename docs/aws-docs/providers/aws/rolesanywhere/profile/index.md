## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
duration_seconds,
enabled,
managed_policy_arns,
name,
profile_arn,
profile_id,
require_instance_properties,
role_arns,
session_policy,
tags
FROM aws.rolesanywhere.profile
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;ProfileId&gt;'
```
