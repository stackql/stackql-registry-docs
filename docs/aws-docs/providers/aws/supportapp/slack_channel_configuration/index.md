## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
<pre>
SELECT * 
FROM aws.supportapp.slack_channel_configuration
WHERE region = 'us-east-1' AND data__Identifier = '&lt;TeamId&gt;' AND data__Identifier = '&lt;ChannelId&gt;'
</pre>
