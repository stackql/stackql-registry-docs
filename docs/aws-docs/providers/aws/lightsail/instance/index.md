## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
support_code,
resource_type,
is_static_ip,
private_ip_address,
public_ip_address,
location,
hardware,
state,
networking,
user_name,
ssh_key_name,
instance_name,
availability_zone,
bundle_id,
blueprint_id,
add_ons,
user_data,
key_pair_name,
tags,
instance_arn
FROM aws.lightsail.instance
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;InstanceName&gt;'
```
