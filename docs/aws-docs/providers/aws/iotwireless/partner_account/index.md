## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
sidewalk,
partner_account_id,
partner_type,
sidewalk_response,
account_linked,
sidewalk_update,
fingerprint,
arn,
tags
FROM aws.iotwireless.partner_account
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;PartnerAccountId&gt;'
```
