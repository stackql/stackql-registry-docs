## `SELECT` Example
```sql
SELECT
region,
arn,
creation_time,
definition_uri,
description,
engine,
id,
main,
name,
parameter_template,
status,
accelerators,
storage_capacity,
tags,
type
FROM aws.omics.workflow
WHERE data__Identifier = '<Id>';
```

## Permissions

To operate on the <code>workflow</code> resource, the following permissions are required:

### Read
```json
omics:GetWorkflow
```

### Update
```json
omics:UpdateWorkflow,
omics:GetWorkflow,
omics:TagResource,
omics:ListTagsForResource,
omics:UntagResource
```

### Delete
```json
omics:DeleteWorkflow,
omics:GetWorkflow
```
