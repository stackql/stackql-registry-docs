## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
node_id,
member_id,
arn,
network_id,
node_configuration
FROM aws.managedblockchain.node
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;NodeId&gt;'
```
