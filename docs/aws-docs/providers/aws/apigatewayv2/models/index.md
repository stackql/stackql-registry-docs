## `SELECT` Example
```sql
SELECT
region,
api_id,
model_id
FROM aws.apigatewayv2.models
WHERE region = 'us-east-1'
```

## Permissions

To operate on the <code>models</code> resource, the following permissions are required:

### Create
```json
apigateway:POST
```

### List
