## `SELECT` Example
```sql
SELECT
region,
route_id,
route_response_selection_expression,
request_models,
operation_name,
authorization_scopes,
api_key_required,
route_key,
authorization_type,
model_selection_expression,
api_id,
request_parameters,
target,
authorizer_id
FROM aws.apigatewayv2.route
WHERE region = 'us-east-1' AND data__Identifier = '<ApiId>|<RouteId>';
```


## Permissions

To operate on the <code>route</code> resource, the following permissions are required:

### Update
```json
apigateway:PATCH,
apigateway:GET,
apigateway:PUT
```

### Read
```json
apigateway:GET
```
