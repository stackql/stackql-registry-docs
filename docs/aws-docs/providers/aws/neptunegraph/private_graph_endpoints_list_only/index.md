## `SELECT` examples
Lists all <code>private_graph_endpoints</code> in a region.
```sql
SELECT
region,
private_graph_endpoint_identifier
FROM aws.neptunegraph.private_graph_endpoints_list_only
WHERE region = 'us-east-1';
```
