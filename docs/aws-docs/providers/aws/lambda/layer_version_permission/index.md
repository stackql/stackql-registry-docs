## Permissions

To operate on the <code>layer_version_permission</code> resource, the following permissions are required:

### Read
```json
lambda:GetLayerVersionPolicy
```

### Delete
```json
lambda:GetLayerVersionPolicy,
lambda:RemoveLayerVersionPermission
```
