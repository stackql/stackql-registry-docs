## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
function_name,
action,
event_source_token,
function_url_auth_type,
source_arn,
source_account,
principal_org_id,
id,
principal
FROM aws.lambda.permission
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;Id&gt;'
```
