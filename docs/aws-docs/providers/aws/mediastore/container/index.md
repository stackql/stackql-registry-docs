## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
policy,
metric_policy,
endpoint,
container_name,
cors_policy,
lifecycle_policy,
access_logging_enabled,
id,
tags
FROM aws.mediastore.container
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;Id&gt;'
```
