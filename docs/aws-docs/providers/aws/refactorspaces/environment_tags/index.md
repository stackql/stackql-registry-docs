## `SELECT` examples
Expands tags for all <code>environments</code> in a region.
```sql
SELECT
region,
description,
environment_identifier,
name,
network_fabric_type,
arn,
transit_gateway_id,
tag_key,
tag_value
FROM aws.refactorspaces.environment_tags
WHERE region = 'us-east-1';
```
