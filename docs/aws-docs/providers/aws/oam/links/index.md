## `SELECT` Example
```sql
SELECT
region,
arn
FROM aws.oam.links
WHERE region = 'us-east-1'
```

## Permissions

To operate on the <code>links</code> resource, the following permissions are required:

### Create
```json
oam:CreateLink,
oam:GetLink,
cloudwatch:Link,
logs:Link,
xray:Link,
applicationinsights:Link,
internetmonitor:Link
```

### List
```json
oam:ListLinks
```
