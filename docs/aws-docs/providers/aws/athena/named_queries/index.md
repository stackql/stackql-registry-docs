## `SELECT` Example
```sql
SELECT
region,
named_query_id
FROM aws.athena.named_queries
WHERE region = 'us-east-1'
```

## Permissions

To operate on the <code>named_queries</code> resource, the following permissions are required:

### Create
```json
athena:CreateNamedQuery
```

### List
