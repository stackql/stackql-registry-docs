## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
monitors,
description,
id,
application_id,
tags,
name
FROM aws.appconfig.environment
WHERE region = 'us-east-1'
AND data__Identifier = '<Id>'
```
