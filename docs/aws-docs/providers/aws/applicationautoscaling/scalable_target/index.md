## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
scheduled_actions,
resource_id,
service_namespace,
scalable_dimension,
suspended_state,
id,
min_capacity,
role_ar_n,
max_capacity
FROM aws.applicationautoscaling.scalable_target
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;Id&gt;'
```
