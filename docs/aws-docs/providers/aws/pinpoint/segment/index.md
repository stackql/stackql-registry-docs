## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
segment_id,
arn,
segment_groups,
dimensions,
application_id,
tags,
name
FROM aws.pinpoint.segment
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;SegmentId&gt;'
```
