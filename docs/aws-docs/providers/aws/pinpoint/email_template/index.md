## Example
```sql
SELECT
region,
html_part,
text_part,
template_name,
template_description,
default_substitutions,
id,
arn,
subject,
tags
FROM aws.pinpoint.email_template
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;Id&gt;'
```
