## Example
```sql
SELECT
region,
id,
bundle_id,
private_key,
enabled,
default_authentication_method,
token_key,
application_id,
team_id,
certificate,
token_key_id
FROM aws.pinpoint.apns_channel
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;Id&gt;'
```
