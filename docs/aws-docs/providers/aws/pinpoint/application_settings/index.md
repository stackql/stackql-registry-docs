## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
id,
quiet_time,
limits,
application_id,
campaign_hook,
cloud_watch_metrics_enabled
FROM aws.pinpoint.application_settings
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;Id&gt;'
```
