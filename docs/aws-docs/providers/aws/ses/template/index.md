## `SELECT` Example
```sql
SELECT
region,
id,
template
FROM aws.ses.template
WHERE data__Identifier = '<Id>';
```

## Permissions

To operate on the <code>template</code> resource, the following permissions are required:

### Read
```json
ses:GetEmailTemplate,
ses:GetTemplate
```

### Delete
```json
ses:DeleteEmailTemplate,
ses:DeleteTemplate
```

### Update
```json
ses:GetEmailTemplate,
ses:UpdateEmailTemplate,
ses:GetTemplate,
ses:UpdateTemplate
```
