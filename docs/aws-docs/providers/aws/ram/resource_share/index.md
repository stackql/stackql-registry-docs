## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
permission_arns,
principals,
allow_external_principals,
id,
arn,
resource_arns,
tags,
name
FROM aws.ram.resource_share
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;Id&gt;'
```
