## `SELECT` examples

Gets all properties from an individual <code>role_policy</code>.
```sql
SELECT
region,
policy_document,
policy_name,
role_name
FROM aws.iam.role_policies
WHERE data__Identifier = '<PolicyName>|<RoleName>';
```

## `DELETE` example

```sql
/*+ delete */
DELETE FROM aws.iam.role_policies
WHERE data__Identifier = '<PolicyName|RoleName>'
AND region = 'us-east-1';
```

## Permissions

To operate on the <code>role_policies</code> resource, the following permissions are required:

### Create
```json
iam:PutRolePolicy,
iam:GetRolePolicy
```

### Read
```json
iam:GetRolePolicy
```

### Update
```json
iam:PutRolePolicy,
iam:GetRolePolicy
```

### Delete
```json
iam:DeleteRolePolicy,
iam:GetRolePolicy
```
