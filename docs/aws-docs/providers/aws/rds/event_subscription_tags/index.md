## `SELECT` examples
Expands tags for all <code>event_subscriptions</code> in a region.
```sql
SELECT
region,
subscription_name,
enabled,
event_categories,
sns_topic_arn,
source_ids,
source_type,
tag_key,
tag_value
FROM aws.rds.event_subscription_tags
WHERE region = 'us-east-1';
```
