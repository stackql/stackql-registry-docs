## `SELECT` Example
```sql
SELECT
region,
arn
FROM aws.ivs.channels
WHERE region = 'us-east-1'
```

## Permissions

To operate on the <code>channels</code> resource, the following permissions are required:

### Create
```json
ivs:CreateChannel,
ivs:TagResource
```

### List
```json
ivs:ListChannels,
ivs:ListTagsForResource
```
