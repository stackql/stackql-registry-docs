## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
status,
user_id,
user_name,
engine,
access_string,
no_password_required,
passwords,
arn,
authentication_mode,
tags
FROM aws.elasticache.user
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;UserId&gt;'
```
