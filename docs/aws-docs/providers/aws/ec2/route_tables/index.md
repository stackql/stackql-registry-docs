## `SELECT` examples
Gets all <code>route_tables</code> in a region.
```sql
SELECT
region,
route_table_id,
vpc_id,
tags
FROM aws.ec2.route_tables
WHERE region = 'us-east-1';
```
Gets all properties from an individual <code>route_table</code>.
```sql
SELECT
region,
route_table_id,
vpc_id,
tags
FROM aws.ec2.route_tables
WHERE region = 'us-east-1' AND data__Identifier = '<RouteTableId>';
```

## `DELETE` example

```sql
/*+ delete */
DELETE FROM aws.ec2.route_tables
WHERE data__Identifier = '<RouteTableId>'
AND region = 'us-east-1';
```

## Permissions

To operate on the <code>route_tables</code> resource, the following permissions are required:

### Read
```json
ec2:DescribeRouteTables
```

### Create
```json
ec2:CreateRouteTable,
ec2:CreateTags,
ec2:DescribeRouteTables
```

### Update
```json
ec2:CreateTags,
ec2:DeleteTags,
ec2:DescribeRouteTables
```

### List
```json
ec2:DescribeRouteTables
```

### Delete
```json
ec2:DescribeRouteTables,
ec2:DeleteRouteTable
```
