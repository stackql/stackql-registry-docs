## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
description,
kms_key_id,
secret_string,
generate_secret_string,
replica_regions,
id,
tags,
name
FROM aws.secretsmanager.secret
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;Id&gt;'
```
