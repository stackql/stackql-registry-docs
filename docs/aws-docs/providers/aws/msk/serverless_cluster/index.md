## Example
```sql
SELECT
region,
arn,
cluster_name,
vpc_configs,
client_authentication,
tags
FROM aws.msk.serverless_cluster
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;Arn&gt;'
```
