## `SELECT` examples
List all <code>config_rules</code> in a region.
```sql
SELECT
region,
config_rule_name
FROM aws.config.config_rules
WHERE region = 'us-east-1';
```
Gets all properties from a <code>config_rule</code>.
```sql
SELECT
region,
config_rule_id,
description,
scope,
config_rule_name,
arn,
compliance,
maximum_execution_frequency,
source,
input_parameters,
evaluation_modes
FROM aws.config.config_rules
WHERE region = 'us-east-1' AND data__Identifier = '<ConfigRuleName>';
```

## `DELETE` example

```sql
/*+ delete */
DELETE FROM aws.config.config_rules
WHERE data__Identifier = '<ConfigRuleName>'
AND region = 'us-east-1';
```

## Permissions

To operate on the <code>config_rules</code> resource, the following permissions are required:

### Create
```json
config:PutConfigRule,
config:DescribeConfigRules
```

### Read
```json
config:DescribeConfigRules,
config:DescribeComplianceByConfigRule
```

### Delete
```json
config:DeleteConfigRule,
config:DescribeConfigRules
```

### List
```json
config:DescribeConfigRules
```

### Update
```json
config:PutConfigRule,
config:DescribeConfigRules
```
