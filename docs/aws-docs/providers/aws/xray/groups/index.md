## `SELECT` Example
```sql
SELECT
region,
group_arn
FROM aws.xray.groups
WHERE region = 'us-east-1'
```

## Permissions

To operate on the <code>groups</code> resource, the following permissions are required:

### Create
```json
xray:CreateGroup,
xray:TagResource
```

### List
```json
xray:GetGroups,
xray:ListTagsForResource
```
