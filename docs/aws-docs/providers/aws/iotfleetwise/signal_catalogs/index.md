## `SELECT` examples
Gets all <code>signal_catalogs</code> in a region.
```sql
SELECT
region,
arn,
creation_time,
description,
last_modification_time,
name,
node_counts,
nodes,
tags
FROM aws.iotfleetwise.signal_catalogs
WHERE region = 'us-east-1';
```
Gets all properties from an individual <code>signal_catalog</code>.
```sql
SELECT
region,
arn,
creation_time,
description,
last_modification_time,
name,
node_counts,
nodes,
tags
FROM aws.iotfleetwise.signal_catalogs
WHERE region = 'us-east-1' AND data__Identifier = '<Name>';
```

## `DELETE` example

```sql
/*+ delete */
DELETE FROM aws.iotfleetwise.signal_catalogs
WHERE data__Identifier = '<Name>'
AND region = 'us-east-1';
```

### Create
```json

iotfleetwise:CreateSignalCatalog,
iotfleetwise:ListSignalCatalogNodes,
iotfleetwise:ListTagsForResource,
iotfleetwise:TagResource
```
