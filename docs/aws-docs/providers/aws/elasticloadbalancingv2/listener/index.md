## Methods
Currently only <code>SELECT</code> is supported for this resource resource.

## Example
```sql
SELECT
region,
ssl_policy,
load_balancer_arn,
default_actions,
port,
certificates,
protocol,
listener_arn,
alpn_policy
FROM aws.elasticloadbalancingv2.listener
WHERE region = 'us-east-1'
AND data__Identifier = '&lt;ListenerArn&gt;'
```
