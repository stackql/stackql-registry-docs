## `SELECT` Example
```sql
SELECT
region,
encryption_configuration,
lifecycle_configuration,
replication_configuration,
meta_store_manager_role_arn,
tags,
arn,
s3_bucket_arn
FROM aws.securitylake.data_lake
WHERE region = 'us-east-1' AND data__Identifier = '<Arn>';
```
