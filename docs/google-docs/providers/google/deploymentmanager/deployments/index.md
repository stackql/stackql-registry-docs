## `SELECT` examples

Lists all deployments for a given project.

```sql
SELECT
id,
name,
description,
fingerprint,
insertTime,
labels,
manifest,
operation,
selfLink,
target,
update,
updateTime
FROM google.deploymentmanager.deployments
WHERE project = '{{ project }}'; 
```

## `UPDATE` example

Updates a deployment only if the necessary resources are available.

```sql
UPDATE google.deploymentmanager.deployments
SET 
id = '{{ id }}',
name = '{{ name }}',
description = '{{ description }}',
operation = '{{ operation }}',
fingerprint = '{{ fingerprint }}',
manifest = '{{ manifest }}',
update = '{{ update }}',
insertTime = '{{ insertTime }}',
updateTime = '{{ updateTime }}',
target = '{{ target }}',
labels = '{{ labels }}',
selfLink = '{{ selfLink }}'
WHERE 
deployment = '{{ deployment }}'
AND project = '{{ project }}';
```

## `DELETE` example

Deletes the specified deployment resource.

```sql
DELETE FROM google.deploymentmanager.deployments
WHERE deployment = '{{ deployment }}'
AND project = '{{ project }}';
```
